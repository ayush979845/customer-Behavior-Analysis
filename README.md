# 📊 Customer Shopping Behavior Analysis

## 🔍 Overview

This project focuses on analyzing customer shopping behavior using Python and Power BI. The goal is to extract meaningful insights from the dataset, understand purchasing patterns, and visualize trends through an interactive dashboard.

---

## 📁 Dataset

The dataset contains customer transaction details, including:

* Customer demographics (Age, Gender)
* Product categories and items purchased
* Purchase amount and frequency
* Payment methods
* Review ratings and subscription status

The data is used to analyze customer preferences and spending behavior.

---

## 🛠️ Tools & Technologies

* **Python** (Pandas)
* **Jupyter Notebook**
* **Power BI**
* **CSV Dataset**

---

## ⚙️ Project Steps

### 1. Data Loading

* Imported dataset using Pandas
* Checked structure using `.head()`, `.info()`, `.describe()`

### 2. Data Cleaning

* Handled missing values
* Removed duplicates
* Converted data types where required

### 3. Exploratory Data Analysis (EDA)

* Analyzed customer demographics
* Studied purchase distribution and trends
* Identified top-performing categories
* Visualized data using charts and graphs

### 4. Feature Understanding

* Examined relationships between variables
* Grouped data based on categories, age groups, and spending

---

## 📊 Dashboard (Power BI)

An interactive Power BI dashboard was created to visualize:

* Total customers and average purchase amount
* Revenue by product category
* Customer segmentation (age, gender)
* Subscription analysis
* Payment method distribution

The dashboard helps in making data-driven business decisions.

---

## 📈 Results & Insights

* Clothing category generates the highest revenue
* Young adults are the most active customers
* Majority of customers are non-subscribers
* Digital payment methods are widely used
* Average purchase amount is around $60
